# 05 — Projects Content

## Featured project: Complete BI project

This is the main proof of capability. It should not be presented as only a screenshot.

Recommended narrative:

```text
The question
What should a decision-maker understand from this data?

The work
Prepared and cleaned the data, structured the analysis, defined KPIs,
and translated the result into an understandable reporting experience.

The output
A complete BI workflow combining data preparation, analysis, KPIs,
visual reporting, and a dashboard designed for clarity.
```

Use real screenshots, schema diagrams, or anonymized samples only. Never expose confidential data.

## YAA Innovations

Show the practical work: Excel correction and preparation, KPI construction, Power BI dashboard work, and a simple interface for presenting information. Keep the description accurate and avoid invented numbers.

## Loveria Café ERP

Present it as a team project demonstrating business software, reporting, dashboards, data flows, and interface consistency. Explain your actual contribution honestly.

## ETL / data work

Show the sequence: source → clean → transform → analyze → communicate. A small animated flow is more useful than a long paragraph.
